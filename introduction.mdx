## Overview

JoinBoundless provides a centralized platform for teams to create, organize, and maintain project documentation. You can build structured knowledge bases, collaborate in real-time, and ensure your docs stay up-to-date with version control and search capabilities. Whether managing technical specs, user guides, or onboarding materials, JoinBoundless streamlines the process.

This platform supports Markdown editing, custom themes, and integrations with tools like GitHub and Slack, making it ideal for developers, product managers, and support teams.

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://app.joinboundless.com` with your email.
  </Step>
  <Step title="New Workspace" icon="plus">
    Create a workspace for your project and invite collaborators.
  </Step>
  <Step title="Add First Doc" icon="file-text">
    Start writing in Markdown. Here's a basic example:
    
    <CodeGroup tabs="Markdown,HTML">
      ```markdown
      # Welcome to My Project
      
      This is your first document.
      
      ## Features
      - Easy editing
      - Live preview
      ```
      ```html
      <h1>Welcome to My Project</h1>
      <p>This is your first document.</p>
      <h2>Features</h2>
      <ul>
        <li>Easy editing</li>
        <li>Live preview</li>
      </ul>
      ```
    </CodeGroup>
  </Step>
  <Step title="Publish" icon="globe">
    Make your docs public or share via link.
  </Step>
</Steps>
