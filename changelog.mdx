## How to Update

Follow these steps to upgrade your JoinBoundless integration to the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command for your platform.
  </Step>
  <Step title="Update Package" icon="download">
    <Tabs>
      <Tab title="npm" icon="package">
````bash
npm install joinboundless@latest
````

        Restart your application after installation.
      </Tab>
      <Tab title="yarn" icon="package">
````bash
yarn add joinboundless@latest
````

        Clear cache if issues persist.
      </Tab>
      <Tab title="Docker" icon="docker">
````dockerfile
FROM joinboundless:latest
````

        Pull and redeploy the image.
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify Update" icon="check-circle">
    <CodeGroup tabs="JavaScript,Python">
```javascript
import { getVersion } from 'joinboundless';
console.log(getVersion()); // Should log "1.2.0"
```
```python
import joinboundless
print(joinboundless.__version__)  # Should print "1.2.0"
```
    </CodeGroup>
  </Step>
</Steps>
