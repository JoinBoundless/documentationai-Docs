---
title: Cannot Access Project
description: How to resolve project access issues.
---

## Symptoms

- "Access Denied" message on project
- Project does not appear in dashboard

## Possible Causes

- You are not a member of the project
- You are logged in with the wrong account

## Quick Fix

<Steps>
  <Step title="Check Login">
    Ensure you are logged in with the correct email for the project.
  </Step>
  <Step title="Request Access">
    Ask your project admin to add you from the project settings.
  </Step>
</Steps>
